# **EP1. 도시 위기**

**“”” 인구가 줄어든 지역에 어떠한 문제가 생기는지, 왜 인구가 감소하는지 이유를 알아보는 것에 포인트를 둘 것 ”””**

# EP1. 지역 위기

## 1. 에피소드 목적

인구 감소와 고령화로 인해 지역에서 어떤 문제가 발생하는지 이해하고, 지역 문제 해결의 필요성을 인식한다.

---

## 2. 핵심 학습 내용

- 지역별 인구 분포의 차이
- 인구 감소 지역의 문제
- 고령화와 생활 기반 시설 감소
- 지역 문제 해결의 필요성

---

## 3. 스토리 배경

가상의 지역 **푸른군**은 한때 사람들이 많이 살고 상권도 활발했던 지역이었지만, 최근 몇 년 사이 인구가 크게 줄어들었다. 젊은 사람들은 일자리를 찾아 다른 지역으로 떠나고, 남아 있는 주민 대부분은 고령층이다. 학교는 학생 수 부족으로 폐교 위기에 놓였고, 시장과 상점도 하나둘 문을 닫고 있다.

학생은 새롭게 임명된 **지역 회복 프로젝트 담당자**가 되어 푸른군의 문제를 살펴보고, 앞으로 어떤 방향으로 지역을 살릴지 고민하게 된다.

---

## 4. 주요 등장인물

| 인물          | 역할                                          |
| ------------- | --------------------------------------------- |
| 군수          | 지역의 현재 상황을 설명하고 프로젝트를 의뢰함 |
| 초등학생 주민 | 학교 폐교 위기와 친구 감소 문제를 이야기함    |
| 시장 상인     | 상권 침체와 손님 감소 문제를 전달함           |
| 고령 주민     | 병원, 교통, 생활 편의 문제를 이야기함         |
| 안내 AI       | 인구 변화 자료와 핵심 개념을 설명함           |

---

## 5. 에피소드 진행 흐름

### 1단계. 도입 컷신

- 낡고 조용한 마을 배경 등장
- 문을 닫은 상점, 빈 운동장, 사람이 적은 거리 표현
- 군수가 학생에게 도움을 요청

핵심 메시지:

> “이 지역은 점점 사람이 줄어들고 있어요. 이대로라면 학교도, 시장도, 마을의 활기도 사라질 수 있습니다.”

---

### 2단계. 지역 탐색

학생이 마을 지도를 살펴보며 주요 장소를 클릭한다.

클릭 장소 예시:

| 장소       | 확인 내용                      |
| ---------- | ------------------------------ |
| 학교       | 학생 수 감소, 폐교 위기        |
| 시장       | 손님 감소, 상점 폐업           |
| 버스정류장 | 교통 불편, 이동 어려움         |
| 보건소     | 고령 주민 증가, 의료 수요 증가 |
| 빈집 지역  | 인구 유출과 주거 공백          |

---

### 3단계. 자료 확인

안내 AI가 간단한 자료를 제시한다.

자료 예시:

- 10년 전 인구: 8,000명
- 현재 인구: 4,200명
- 65세 이상 인구 비율 증가
- 초등학생 수 감소
- 빈집 증가

표현 방식:

- 막대그래프
- 지도 색 변화
- 짧은 설명 카드

“””

단순히:

- “인구가 줄었어요”

가 아니라,

👉 지역 감소 vs 수도권 집중을 함께 보여주는 것

이건 교육과정적으로도 훨씬 좋습니다.

예:

- 수도권 인구 증가 그래프
- 지역 인구 감소 그래프
- 연령대 변화 그래프

👉 “지역 불균형” 개념까지 자연스럽게 연결됨

“””

### 4단계. 문제 원인 생각하기

학생에게 간단한 선택형 질문을 제시한다.

예시 질문:

1. 일자리와 생활 편의 시설이 부족하기 때문이다.
2. 사람들이 모두 여행을 떠났기 때문이다.
3. 지역의 날씨가 매일 나쁘기 때문이다.

정답: 1

다음 목표:

- 지역 경제를 다시 활성화하기

# 🎮 EP3. 경제 성장

---

# 1. 에피소드 목적

경제 성장이 지역 발전에 어떤 영향을 주는지 이해하고, 일자리와 산업 시설이 지역 활성화에 미치는 효과를 경험한다.

---

# 2. 핵심 학습 내용

- 경제 성장의 의미

---

# 3. 스토리 배경

생활 환경이 조금씩 개선되면서 푸른군에는 다시 사람들이 찾아오기 시작했다. 하지만 여전히 지역에는 충분한 일자리가 부족하다.

학생은 이제 푸른군의 경제를 활성화하기 위한 산업과 시설을 고민하게 된다.
